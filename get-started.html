Get Started with CHMER

Step-by-step instructions to install CHMER and run your first program.

Install on Linux

git clone https://github.com/HSR-projects/CHMER-chess-programing-language.git
cd CHMER
./install.sh

Install on Windows

Download CHMER-Windows.zip from GitHub
Extract and run CHMER.exe

First Program

func main() {
  print("Hello CHMER!")
}
